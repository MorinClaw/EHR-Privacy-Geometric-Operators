# A.6 Temporal structure (ACF/PACF, Ljung–Box, spectrum)

Compares **raw vs perturbed** time series structure: ACF/PACF, Ljung–Box tests, and Welch PSD for representative variables.

## Prerequisites

A.2 outputs and aligned `ts_48h` inputs.

## Run

```bash
cd A6_temporal_structure/code
python3 exp_a6_temporal.py
```

Outputs: tables under `results/tables/`, figures under `results/figs/`.
