# ts_48h_weak (generated data)

CSV matrices in this directory are **not committed** to git (see root `.gitignore`).

Generate them locally from your raw pipeline, e.g. run:

```bash
python P3_weak_interpolation/code/build_ts48h_weak.py --help
```

Use paths appropriate for your machine; keep weak time series outside the public repo if they are derived from restricted data.
